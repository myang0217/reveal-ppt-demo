为什么是竖的

画布尺寸就是配置里的两个数字,内容会自动缩放去填满屏幕:

Reveal.initialize({
  width:  402,   // iPhone 16 Pro
  height: 874,   // 逻辑分辨率
});
改成 1280 × 720 就变回投影仪横版。同一份内容,换个画布即可 —— 手机竖屏刷、投影横屏讲。

LaTeX 是一等公民

正文直接写 $...$、$$...$$,KaTeX 负责渲染。

Kohn–Sham 方程:

$$\left[-\frac{\hbar^2}{2m}\nabla^2 + V_{\text{eff}}\right]\psi_i = \varepsilon_i\,\psi_i$$

其中 $V_{\text{eff}} = V_{\text{ext}} + V_{\text{H}} + V_{\text{xc}}$, 而 $V_{\text{xc}} = \delta E_{\text{xc}}[n]/\delta n$。

电子–声子矩阵元: $$g_{mn\nu} = \sqrt{\tfrac{\hbar}{2\omega_{\mathbf{q}\nu}}}\, \langle \psi_{m\mathbf{k+q}} \mid \partial_{\mathbf{q}\nu} V \mid \psi_{n\mathbf{k}} \rangle$$

公式和正文是同一套 LaTeX,不用截图粘贴,改公式就是改文本。
