reveal.js 是怎么回事

用一份 HTML 文件做的幻灯片
—— 这一页本身就是它的产物

这是演讲者备注,按 S 键会弹出单独的演讲者视图。

一句话:幻灯片 = 网页

没有 .pptx,只有一个 index.html。每一页是一个 <section>:

<div class="slides">
  <section>
    <h2>第一页</h2>
  </section>
  <section>
    <h2>第二页</h2>
  </section>
</div>

↑ 代码块可以逐段高亮(这一段就演示了三次高亮跳转,继续按空格)

data-line-numbers 用竖线分隔,每按一次前进就换一组高亮行。

导航是二维的

左右 = 章节之间;上下 = 同一章节内部展开。

右下角出现了 ↓ 箭头 —— 现在按向下键,而不是空格。

主线讲给所有人听,细节埋在向下的分支里。听众问细节时才下钻,不问就直接右移跳过。

LaTeX 公式是一等公民

直接写 $...$ 和 $$...$$,KaTeX 负责渲染:

Kohn–Sham 方程:

$$\left[-\frac{\hbar^2}{2m}\nabla^2 + V_{\text{eff}}(\mathbf{r})\right]\psi_i(\mathbf{r}) = \varepsilon_i\,\psi_i(\mathbf{r})$$

其中有效势 $V_{\text{eff}} = V_{\text{ext}} + V_{\text{H}} + V_{\text{xc}}$,交换关联势 $V_{\text{xc}} = \delta E_{\text{xc}}[n]/\delta n(\mathbf{r})$。

电子–声子矩阵元: $$g_{mn\nu}(\mathbf{k},\mathbf{q}) = \sqrt{\frac{\hbar}{2\omega_{\mathbf{q}\nu}}}\, \langle \psi_{m\mathbf{k}+\mathbf{q}} \mid \partial_{\mathbf{q}\nu} V \mid \psi_{n\mathbf{k}} \rangle$$

公式和正文是同一套 LaTeX,不用截图粘贴,改公式就是改文本。

整页背景随手换

纯色、渐变、图片、视频、iframe 都行:

<section data-background-color="#0b1d33">
<section data-background-image="band.png">
<section data-background-video="md.mp4" data-background-video-loop>
<section data-background-iframe="https://example.org">

分子动力学轨迹、能带演化动画可以直接当背景铺满整页。

现在就能试的快捷键

  • Esc / O — 鸟瞰全部幻灯片(网格视图)
  • S — 演讲者视图:备注 + 计时 + 下一页预览(投影时留给自己看)
  • F — 全屏;B — 黑屏(把注意力拉回你身上)
  • Alt + 点击 — 局部放大某处
  • ? — 列出所有快捷键
要 PDF? 在网址后加 ?print-pdf,再用浏览器打印 → 另存为 PDF。 投影仪没网、会场断网时的保险方案。

怎么改成你自己的

  1. 在 GitHub 上编辑 index.html(直接网页里点铅笔图标就行)
  2. 照着抄一个 <section> 块,换掉里面的字
  3. Commit → 等约 1 分钟,链接上的内容自动更新

换主题:改 <head> 里那一行的文件名即可 ——

theme/white.css   <!-- 当前 -->
theme/black.css   theme/league.css   theme/serif.css
theme/moon.css    theme/solarized.css   theme/dracula.css

就这么回事

一个 HTML 文件 + 一个 GitHub 仓库 = 一份能用链接分享、能 git 回溯的幻灯片。
